Wallets

Desktop Wallets

Bitcoin Only

  • Electrum (opens in a new tab)

    A lightweight and widely used Bitcoin only wallet for desktop users.

Multichain

  • Exodus (opens in a new tab)

    A beginner friendly wallet supporting Bitcoin, Ethereum, and many other assets.

  • Atomic Wallet (opens in a new tab)

    A multichain desktop wallet with built in swapping features.

Browser Wallets

MetaMask

The most commonly used browser wallet for Ethereum and Ethereum based chains.

Trust Wallet

A popular wallet available as a browser extension and mobile app.

Rabby

A browser wallet focused on safety warnings and transaction clarity.

Most wallets also offer mobile apps.

Hardware Wallets

Trezor

A well known hardware wallet for long term secure storage.

Ledger

A popular hardware wallet supporting a wide range of assets.

Tangem

A card based hardware wallet designed for simplicity.

TMX Global

A hardware wallet solution with additional enterprise focused features.

Important:
When purchasing a hardware wallet, always order directly from the manufacturer. Do not buy from third party marketplaces such as Amazon or Newegg.

Swap Tools
