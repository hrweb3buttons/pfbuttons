NextGen Testnet Setup

Use these tools to add the BNB Smart Chain Testnet to your MetaMask wallet and import the NextGen (PFNG) test token.

The testnet is a safe environment where you can experiment without using real funds. The PFNG token here has no real monetary value.

Step 1 — Add BNB Smart Chain Testnet

This will add the BNB Smart Chain Testnet network to your MetaMask. You will need it before you can interact with the NextGen token.

MetaMask will prompt you to approve adding the network. Once added, you may need to switch to it manually from MetaMask's network selector.

Follow these steps to add the testnet manually in MetaMask mobile:

  1. Open MetaMask and tap the menu icon (☰) at the top left.
  2. Tap Settings.
  3. Tap Networks.
  4. Tap Add a network.
  5. Tap Add a network manually at the bottom.
  6. Enter the network details below, then tap Save.

Step 2 — Add NextGen (PFNG) Token

Once you are on the BNB Smart Chain Testnet, use this to import the NextGen token into your wallet so your balance is visible.

MetaMask will ask you to confirm adding the token. Make sure you are connected to the BNB Smart Chain Testnet before importing.

Follow these steps to import the PFNG token in MetaMask mobile:

  1. Make sure you are switched to the BNB Smart Chain Testnet network you just added.
  2. Tap Tokens on the main wallet screen.
  3. Tap the + icon near the top right.
  4. Tap Custom token.
  5. Paste the contract address below into the Token contract address field.
  6. The symbol and decimals should fill in automatically. Tap Import.
